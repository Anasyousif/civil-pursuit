(inner) getCollapsedItem(event) → {Object}

Get information about Navigator Item

This function gets triggered whenever an item is expanded or collapsed, and returns the following information about the target item as an Object with the following properties:

  • Object is - an array made of a single ItemSchema
  • String type - the item's plural type (ie, topics, problems, etc)
  • String has - the item's children's plural type (ie, problems, opinions, etc)
  • String id - the item's objectId's string
